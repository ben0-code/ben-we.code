Présentation du projet

Ce projet consiste à réaliser un web scraping du site CoinAfrique afin de collecter automatiquement les annonces de PlayStation 5 (PS5).
Le script est développé en Python en utilisant les bibliothèques Requests, BeautifulSoup et Pandas.

Il permet d’extraire des informations comme le nom de l’article, le prix, la localisation et la date de publication, puis de les organiser dans un DataFrame pour faciliter l’analyse des données.

L’objectif est de suivre et analyser les prix des PS5 sur le marché en ligne en Côte d’Ivoire.
